## Sobre 

O site da **Gym Buddy**, feito para o TCC do curso de Desenvolvimento de Sistemas, na escola técnica SENAI Jandira, permite o usuário ter uma experiência de treinos totalmente personalizada. Com o uso de um Agente IA, a montagem de treinos e análise de resultados é mais precisa, tornando o uso da aplicação mais interativo para o usuário final.
